# PyroGuards Internship - Login Screen with Flutter UI

## Overview

This repository contains the work I completed during my internship at **PyroGuards**. The primary focus was to create a **Login Screen** using the **Flutter UI package**. This project also includes some custom widgets and animations that I implemented to enhance the user experience.

## Project Features

- **Login Screen UI**: Designed a responsive and modern login page using Flutter.
- **Flutter UI Package**: Used to streamline the creation of a clean and visually appealing interface.
- **Custom Animations**: Added smooth animations to enhance the user experience.
- **Custom Widgets**: Created reusable widgets for buttons and input fields.
- **Clean and Efficient Code**: Followed best practices to keep the code modular, readable, and maintainable.

## Technologies Used

- **Flutter**: The primary framework used for building the login screen.
- **Dart**: The programming language for Flutter development.
- **Flutter UI Package**: To design the user interface.
- **Custom Animations**: To make the UI elements more interactive and user-friendly.

## Installation

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/gowtham-dd/Intern-pyro.git
   ```

2. Navigate to the project folder:

   ```bash
   cd login_screen2
   ```

3. Install the necessary dependencies:

   ```bash
   flutter pub get
   ```

4. Run the app on an emulator or a physical device:

   ```bash
   flutter run
   ```
